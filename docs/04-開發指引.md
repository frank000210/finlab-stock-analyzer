# 開發指引

> 給接手 / 協作 **finlab-stock-analyzer** 的工程師（含 AI）的實務指引。
> 搭配 `03-技術架構.md` 理解結構，`00-開發流程.md` 理解流程。

---

## 一、環境準備

### 前置需求
- Node.js 20+、Python 3.11、git；（部署用）Zeabur CLI（`npx zeabur`）。
- 後端原生相依 TA-Lib：本機若要跑技術分析需另裝；部署由 Dockerfile 編譯。

### 後端
```powershell
cd backend
python -m venv .venv; .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
# 複製 .env.example → .env，填入 FINMIND_TOKEN / GOOGLE_CLIENT_ID / MONGODB_URI 等
uvicorn app.main:app --reload --port 8000
```
- MongoDB 非必須：未連線時自動降級（快取/log 略過），API 仍可運作。
- 健康檢查：`curl http://localhost:8000/api/health`。
- API 文件：`http://localhost:8000/api/docs`（Swagger）。

### 前端
```powershell
cd frontend
npm install
npm run dev     # http://localhost:5173，API_BASE 自動指向 localhost:8000
```

## 二、本機開發流程（每個增量）

1. **先讀**：相關 view / api / analysis 與 `assets/main.css` 設計 token，沿用既有風格。
2. **小步實作**：一次一個可驗證增量；文案用繁體中文。
3. **本地驗證**（缺一不可）：
   ```powershell
   cd frontend; npm run build                                  # 前端建置須過
   cd ..\backend; python -c "from app.main import app; print('OK')"   # 後端匯入須過
   ```
4. build / import 不過，**不要 commit**。

## 三、後端開發約定

### 新增一個 API 端點
1. 在 `analysis/`（或對應層）寫**純運算函式**（可單測、不碰 HTTP）。
2. 在 `api/<feature>.py` 建 `APIRouter(prefix="/api/v1/...")`，呼叫運算層。
3. 回傳統一格式 `{"success": True, "data": ...}`。
4. 在 `api/__init__.py` 匯出 router，於 `main.py` `include_router`。
5. 重運算請用快取：
   ```python
   cache_key = f"<feature>:v1:{symbol}:{days}"   # 改演算法時 bump 版號
   ```
6. 多來源平行抓取用 `asyncio.gather(..., return_exceptions=True)`，逐一判斷例外。

### 注意事項
- 金融資料常含 NaN / numpy 型別 → 已由 `SafeJSONResponse` 統一清洗，但運算層也盡量回原生型別。
- 祕密一律走 `settings`（環境變數），**不可硬編碼**。
- 失敗要降級不要整碗端走（單一資料源掛掉，其他面向仍要能呈現）。

## 四、前端開發約定

### 新增一個頁面
1. 在 `views/` 建 `XxxView.vue`，於 `router.js` 以 `() => import()` 懶載入註冊。
2. 取數用頁內 `apiGet(path)`；務必 **驗證回傳形狀** 再使用：
   ```js
   const payload = await apiGet(`/api/v1/stocks/${symbol}/chip-score`)
   if (payload && typeof payload === 'object' && 'score' in payload) { /* 用它 */ }
   ```
3. 樣式只用 `main.css` 的 CSS 變數；數字加 `font-variant-numeric: tabular-nums`。
4. 漲跌色用 `--color-up / --color-down`；不要自訂硬色碼。
5. 圖表用 `lightweight-charts`；K 線疊圖參考 `AnalysisView.vue` / `ChipAnalysisView.vue`。
6. 狀態用 Pinia：`useStockStore().setStock(sym, name)`（`symbol/name` 唯讀）。

### 進場動效（v-reveal）
```html
<section v-reveal>...</section>
<article v-reveal="{ delay: index * 70 }">...</article>   <!-- stagger 交錯 -->
```
- 規則：**內容預設可見**，動效只是增強；指令已含 1.4s failsafe 與 `prefers-reduced-motion` 後備。
- 不可把「是否顯示」綁在動畫 class 上（否則隱藏分頁 / headless 會留白）。

### 設計禁忌（避免 AI 套版味）
- 不要等大卡片網格、漸層文字、側邊色條（>1px 彩色 border-left/right）。
- 不要每段都加小寫追蹤字距的 eyebrow 小標。
- 用層次（bento、大小對比）與真實資訊架構取代重複堆疊。

## 六、部署（重點，務必照做）⚠️

```powershell
cd F:\github\finlab-stock-analyzer        # 一定要 repo 根目錄！
$env:ZEABUR_TOKEN="sk-..."
npx zeabur deploy --project-id 6a3f6ff522d1fdaf7eb04d04 --service-id 6a3f700322d1fdaf7eb04d06 -i=false
```

**為何一定要根目錄**：服務用根 `Dockerfile` 建「前後端合併映像」。
- 從子目錄 / 空目錄部署 → 被判為純前端 static → 全站 404 或 `/api` 失聯。
- 正確 docker build 會編譯 TA-Lib，約 **7-8 分鐘**；若只花 ~1 分鐘 = 跑成 static（錯誤）。

**狀態 / 日誌**：
```powershell
npx zeabur deployment list --service-id <SID> -i=false --json      # status / planType（要 docker）
npx zeabur deployment log  --deployment-id <DID> --service-id <SID> --type build   -i=false
npx zeabur deployment log  --deployment-id <DID> --service-id <SID> --type runtime -i=false
```

## 七、線上驗證（部署後必跑）

| 檢查 | 指令 | 通過條件 |
|---|---|---|
| 首頁 | `curl https://<domain>/` | HTTP 200 且 index hash 與前一版不同 |
| 後端連通 | `curl -D - https://<domain>/api/v1/stocks/2330/info` | `Content-Type: application/json`（**不是** text/html） |
| 新功能 chunk | `curl https://<domain>/assets/<Chunk>.js` | 200 且 grep 得到新標記字串 |
| 互動 | 瀏覽器 MCP 開頁 | console 無錯、互動正常 |

> 截圖只顯示給使用者、AI 看不到 → AI 要用 `read_page` / `evaluate_javascript` 抓 DOM 與資料驗證。

## 八、除錯常見問題

| 症狀 | 可能原因 | 處理 |
|---|---|---|
| 全站 404 / 空白 | 從錯誤目錄部署成 static | 改從 repo 根目錄重新部署 |
| `/api` 回 HTML | 部署成純前端、後端失聯 | 同上，確認 planType=docker |
| 前端解析 JSON 失敗 | 直接用了 `payload.data` 但 data 為 null | 先驗證形狀（`'key' in payload`） |
| 改了演算法但結果沒變 | 命中舊快取 | bump cache key 版號（`:v5`→`:v6`） |
| 數字顯示 NaN | numpy/NaN 未清洗 | 走 `SafeJSONResponse`；運算層回原生型別 |
| 動效後頁面留白 | 把顯示綁在動畫 class | 用 `v-reveal`（預設可見 + failsafe） |
| 圖表不顯示 | 容器無高度 / 資料空 | 給固定高度、資料空時顯示佔位 |
